⚡ DÉMARRAGE EN 3 ÉTAPES
═══════════════════════════════════════════════════════════════

ÉTAPE 1️⃣: INSTALLATION BD
───────────────────────────────────────────────────────────────
1. Ouvrez phpMyAdmin (http://localhost/phpmyadmin)
2. Créez une base de données: gestion_centre_formation
3. Importez le fichier: gestion_centre_formation.sql
4. (Optionnel) Importez: donnees_exemple.sql pour les données de test

ÉTAPE 2️⃣: VÉRIFICATION
───────────────────────────────────────────────────────────────
1. Allez sur: http://localhost/Gestion_Formation/check-install.php
2. Vérifiez que tous les éléments sont ✓ OK
3. Corrigez les erreurs si nécessaire

ÉTAPE 3️⃣: DÉMARRAGE
───────────────────────────────────────────────────────────────
1. Allez sur: http://localhost/Gestion_Formation/
2. Explorez la landing page
3. Cliquez sur "Tableau de Bord" pour commencer


🎯 PREMIERS PAS
═══════════════════════════════════════════════════════════════

Après le démarrage, essayez ces actions:

1. AJOUTER UN APPRENANT
   Dashboard → Apprenants → Ajouter Apprenant
   
2. CRÉER UNE FILIÈRE
   Dashboard → Filières → Ajouter Filière
   
3. CRÉER UNE INSCRIPTION
   Dashboard → Inscriptions → Nouvelle Inscription
   
4. ENREGISTRER UN PAIEMENT
   Dashboard → Paiements → Enregistrer Paiement
   
5. VOIR LES STATISTIQUES
   Dashboard → Statistiques (menu latéral)

🔧 CONFIGURATION
═══════════════════════════════════════════════════════════════

Si la connexion BD échoue:

1. Ouvrez: config/Database.php
2. Modifiez les identifiants:
   - $host = 'localhost'
   - $user = 'root'
   - $password = '' (ou votre mot de passe)
   - $db_name = 'gestion_centre_formation'
3. Sauvegardez et rechargez la page

🆘 PROBLÈMES COURANTS
═══════════════════════════════════════════════════════════════

❌ Page blanche au démarrage
   ✓ Vérifiez check-install.php
   ✓ Vérifiez la connexion BD
   ✓ Consultez les logs Apache

❌ Données qui ne s'affichent pas
   ✓ Vérifiez que la BD contient des données
   ✓ Actualisez (Ctrl+F5)
   ✓ Vérifiez la console (F12)

❌ Impossible de se connecter à la BD
   ✓ Vérifiez que MySQL est en cours d'exécution
   ✓ Vérifiez les identifiants dans config/Database.php
   ✓ Vérifiez que la BD existe

❌ Les modals ne s'ouvrent pas
   ✓ Videz le cache (Ctrl+Maj+Del)
   ✓ Vérifiez la console (F12)
   ✓ Actualisez la page
